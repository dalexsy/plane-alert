Movement Direction Test

For a plane moving right (deltaX > 0), it should rotate 90° to point right.

For a plane moving left (deltaX < 0), it should rotate 270° (or -90°) to point left.

For a plane moving up (deltaY > 0), it should rotate 0° to point up.

For a plane moving down (deltaY < 0), it should rotate 180° to point down.
